Description:

Directory containing 4 folders as 4 population(EUR for European, EAS for East Asian, AFR for African, AMR for American), each folder containing files for each chromosome. Each file contains precomputed R² values for pairs of SNPs. A line is tab-delimited and formatted as: SNP1, SNP2, R² value. The threshold of R² value is 0.05. Mentioned that the positions of SNPs are 1-based.
