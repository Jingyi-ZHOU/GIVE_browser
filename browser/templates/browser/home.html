Introduction

Welcome to PARSE.
PARSE is an academic website. It is an excellent tool with three functional sections. Users can get informative message from several aspects.

Functions

1. Data

Data section is feasible for achieving an overall statistical impression of the data used to generating this website. Users can dynamically visualize the statistical features of different variants. It also offers descriptions and download links of all data sets.

2. Find LD SNPs

Find LD SNPs section is created for searching linked SNPs based on user specific SNPs or exsiting GWAS. Users can select linked SNPs by setting RÂ² and specific population.

Visualization section provides interactive genome browser to view GWAS and LD SNPs. Moreover, genes, RDDAR, PhastCons data are also available in the visualization part.
